# Pinsora Auto Image Generator

سيرفر Python يولد صور تلقائياً لكل فئة في الموقع.

## كيف يعمل

1. يتصل بقاعدة بيانات Supabase
2. يجلب كل الفئات الرئيسية (15 فئة)
3. كل دقيقة يولد صورة واحدة لفئة تحتاج صور
4. الهدف: 5 صور لكل فئة = 75 صورة إجمالاً
5. يحفظ الصورة مباشرة في قاعدة البيانات

## التثبيت

```bash
cd image-generator
pip install -r requirements.txt
```

## التشغيل

```bash
python generator.py
```

## الإعدادات

في ملف `generator.py`:
- `TARGET_IMAGES_PER_CATEGORY` = عدد الصور المطلوبة لكل فئة (افتراضي: 5)
- `INTERVAL_SECONDS` = الوقت بين كل صورة بالثواني (افتراضي: 60)

## الوقت المتوقع

- 15 فئة × 5 صور = 75 صورة
- كل صورة = دقيقة واحدة
- الإجمالي = ~75 دقيقة
